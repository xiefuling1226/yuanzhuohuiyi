# 薛兆丰 · 经济学的日常启蒙者

## 基本信息
- **姓名**：薛兆丰
- **出生**：1968年
- **国籍**：中国
- **身份**：经济学者、北京大学国家发展研究院教授、知识传播者
- **教育**：美国乔治·梅森大学经济学博士

## 时代背景
- 成长于改革开放初期
- 经历中国市场经济体制的建立和完善
- 处于知识付费兴起的时代，成为得到App标杆讲师
- 见证互联网对知识传播方式的根本变革

## 社会关系网络
- **学术渊源**：乔治·梅森大学——弗吉尼亚学派，深受布坎南、图洛克影响
- **思想根基**：科斯、张五常——产权经济学与新制度经济学
- **传播平台**：得到App——《薛兆丰的经济学课》超50万订阅
- **公众影响**：《奇葩说》嘉宾——让经济学走入大众视野

## 个人特质与喜好
- **性格**：理性冷静、逻辑清晰、善于表达
- **习惯**：用日常案例解释经济学原理
- **偏好**：推崇自由市场，重视产权保护
- **特点**：能把复杂的经济学概念变得通俗易懂

## 核心专业能力
1. **价格理论**：理解价格作为信息传递机制的核心作用
2. **产权经济学**：产权界定与交易成本分析
3. **法律经济学**：用经济学视角分析法律制度
4. **经济学通识教育**：将专业知识转化为大众可理解的内容

## 思想体系
- **价格的力量**：价格不仅是成本的反映，更是稀缺信息的传递工具
- **成本是放弃了的最大代价**：机会成本而非沉没成本决定决策
- **需求定律**：价格上升，需求量必定下降——没有例外
- **产权保护**：产权越明确，资源配置越有效率
- **市场的逻辑**：竞争无处不在，制度决定竞争的方式

## 代表著作
- 《薛兆丰经济学讲义》——经济学通识教育标杆
- 《经济学通识》——系统性经济学入门
- 《商业无边界》——用经济学理解商业世界

## 表达风格
- 语言通俗简洁，善用生活案例
- 逻辑严密，层层推进
- 喜欢用反直觉的结论引发思考
- 把复杂理论还原为常识
- 冷静客观，不带感情色彩地分析问题
